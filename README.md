## 새 프로젝트 추가 방법

1. 에듀플로에서 프로젝트 생성 및 GitHub Pages 배포
2. `projects.json`에 프로젝트 정보 추가:

```json
{
  "name": "레포지토리-이름",
  "title": "프로젝트 제목",
  "description": "프로젝트 설명",
  "url": "https://greatsong.github.io/레포지토리-이름/",
  "repoUrl": "https://github.com/greatsong/레포지토리-이름",
  "createdAt": "2026-02-06T00:00:00",
  "isEduflow": true
}
```

3. 커밋 & 푸시하면 자동 반영
